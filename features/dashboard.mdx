The MemoryPlugin Dashboard is your central hub for managing memories and buckets. Access it at [memoryplugin.com/dashboard](https://memoryplugin.com/dashboard).


## Dashboard Overview

The dashboard has a tabbed interface with four main sections:

<CardGroup cols={2}>
  <Card title="Memories" icon="brain">
    View, search, and manage all your memories. Switch between buckets and organize with Smart Memory categories.
  </Card>
  <Card title="Memory Chat" icon="chat">
    Chat directly with your memories to ask questions and get AI-powered responses.
  </Card>
  <Card title="Suggestions" icon="lightbulb">
    AI-powered suggestions to optimize your memories by deleting duplicates, combining similar ones, or updating outdated information.
  </Card>
  <Card title="Experiments" icon="flask">
    Smart Memory Builder - have a conversation with AI to quickly populate your memory system instead of adding memories manually.
  </Card>
</CardGroup>

## Core Functionality

### Managing Your Memories
- **Browse memories**: View all memories in a searchable list
- **Search**: Find specific memories using the search bar
- **Edit memories**: Click to modify memory text
- **Delete memories**: Remove individual or multiple memories
- **Bulk operations**: Select multiple memories for batch actions

### Organizing with Buckets
- **Create buckets**: Click "New Bucket" to create different contexts
- **Switch buckets**: Use the bucket selector to change contexts
- **Move memories**: Transfer memories between buckets
- **Smart Categories**: When you have 30+ memories, create AI-organized categories

### Getting Your Auth Token
Your Auth Token is needed for integrations like MCP Server and Custom GPT.

<Warning>
Keep your Auth Token confidential. Regenerate it if accidentally exposed.
</Warning>

1. Sign in to your dashboard
2. Click "Settings" in the navigation or "Go to Settings" button
3. Go to the "Integration & Access" section  
4. Copy your Auth Token
