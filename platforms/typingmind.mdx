## Integration Options

### TypingMind Plugin

Install the official MemoryPlugin for TypingMind:

[Install TypingMind Plugin](https://cloud.typingmind.com/plugins/p-01JFW700XGDXQ2WPJPW2JXN8MX)

See the [TypingMind Plugin guide](/integrations/typingmind-plugin) for setup instructions.

### Browser Extension

1. Install the [MemoryPlugin browser extension](/integrations/browser-extension)
2. Navigate to [typingmind.com](https://typingmind.com)
3. Click the MemoryPlugin button to inject memories
4. Start chatting with memory context
