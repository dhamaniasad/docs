# Supported Platforms

MemoryPlugin supports a growing list of AI platforms, allowing you to maintain consistent memory across all your favorite AI tools.

## Platform Detection

The browser extension automatically detects supported platforms by:

1. **URL Matching**: Identifying platform by domain
2. **DOM Detection**: Recognizing platform-specific UI elements
3. **Dynamic Loading**: Adapting to platform interface changes

## Platform-Specific Features

Different platforms support varying MemoryPlugin features:

### Full Feature Support
Platforms with complete MemoryPlugin integration:
- Memory creation and retrieval
- Bucket switching
- Smart Memory support
- Import/export capabilities

### Basic Support
Platforms with core functionality:
- Memory injection
- Basic memory creation
- Default bucket usage

### Custom Implementations
Platforms with specialized integrations:
- **ChatGPT**: Custom GPT with function calling
- **Claude**: Native MCP server support
- **TypingMind**: Dedicated plugin system

## Troubleshooting Platform Issues

<AccordionGroup>
  <Accordion title="Platform not detected">
    - Refresh the page
    - Check extension is enabled
    - Verify you're on a supported URL
    - Update to latest extension version
  </Accordion>
  <Accordion title="Memory injection fails">
    - Check platform didn't update interface
    - Try different injection method (button vs. quick inject)
    - Report issue with platform details
  </Accordion>
  <Accordion title="Platform-specific features missing">
    - Some features are platform-dependent
    - Check integration method compatibility
    - Consider alternative integration method
  </Accordion>
</AccordionGroup>
