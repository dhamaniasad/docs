Use MemoryPlugin with Grok at [grok.com](https://grok.com).

## How to Use

### Browser Extension
1. Install the [MemoryPlugin Browser Extension](/integrations/browser-extension)
2. Navigate to [grok.com](https://grok.com)
3. Click the MemoryPlugin button to inject memories
4. Start chatting with memory context

## Chat History

You can import your Grok conversation history into MemoryPlugin:

- **File upload**: Export your Grok data and upload the conversation file to MemoryPlugin
- **Online sync**: Sync conversations directly from the browser extension

Once imported, your past Grok conversations become searchable and available as context across all your AI tools.

See the [Chat History](/features/chat-history/introduction) docs for more details.
