Use MemoryPlugin with ChatGPT at [chat.openai.com](https://chat.openai.com) or [chatgpt.com](https://chatgpt.com).

## How to Use

### Browser Extension (Recommended)
1. Install the [MemoryPlugin Browser Extension](/integrations/browser-extension)
2. Go to ChatGPT
3. Click the MemoryPlugin button to inject memories
4. Start chatting with memory context

### Custom GPT (Alternative)
Use the [MemoryPlugin Custom GPT](https://chatgpt.com/g/g-pPRwrR7LI-memoryplugin) - memories load automatically.

### Add to Your Own GPT
Follow the [Custom GPT Integration](/integrations/custom-gpt-integration) guide to add memory to your own GPTs.

## Special Features

### Import ChatGPT Memories
Transfer your existing ChatGPT memories to MemoryPlugin:
1. Go to ChatGPT Settings → Personalization → Manage memories
2. Click the "Sync Memories" button (MemoryPlugin logo)
3. Wait for import to complete
4. Check your [Dashboard](https://memoryplugin.com/dashboard) to see imported memories

<Info>
Keep the browser tab open during sync - closing it will interrupt the process.
</Info>
