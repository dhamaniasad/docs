Use MemoryPlugin with DeepSeek at [chat.deepseek.com](https://chat.deepseek.com).

## How to Use

### Browser Extension
1. Install the [MemoryPlugin Browser Extension](/integrations/browser-extension)
2. Navigate to [chat.deepseek.com](https://chat.deepseek.com)
3. Click the MemoryPlugin button to inject memories
4. Start chatting with memory context

## Chat History

You can sync your DeepSeek conversation history into MemoryPlugin using the browser extension's online sync feature. Once synced, your past DeepSeek conversations become searchable and available as context across all your AI tools.

See the [Chat History](/features/chat-history/introduction) docs for more details.
