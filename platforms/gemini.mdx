# Gemini Integration

Google Gemini is a powerful multimodal AI that works seamlessly with MemoryPlugin through the browser extension, bringing persistent memory to your AI conversations.

## Integration Method

<Card title="Browser Extension" icon="browser" href="/integrations/browser-extension">

## Quick Start

<Steps>
  <Step title="Install Extension">
    Install the [MemoryPlugin Browser Extension](/integrations/browser-extension) for Chrome or other Chromium browsers
  </Step>
  <Step title="Navigate to Gemini">
    Go to [gemini.google.com](https://gemini.google.com)
  </Step>
  <Step title="Sign In">
    Log in to both Gemini and MemoryPlugin accounts
  </Step>
  <Step title="Activate MemoryPlugin">
    Click the MemoryPlugin button to inject your memories
  </Step>
  <Step title="Start Chatting">
    Gemini now has access to your memory context
  </Step>
</Steps>

## Using MemoryPlugin with Gemini

### Memory Injection Process

1. **Locate the Button**: Look for the MemoryPlugin button (bottom-right or near input)
2. **Click to Activate**: Injects memories into the conversation
3. **Visual Confirmation**: See memories added to chat context
4. **Ready to Use**: Gemini processes memories for the conversation

### Creating Memories in Gemini

Tell Gemini to remember information:
- "Remember that I'm working on [project name]"
- "Add to memory: My preferred coding language is Python"
- "Save this for later: [important information]"

The extension detects memory creation patterns:
```
to=memoryplugin&&memory=[Your memory content]
```

### Managing Buckets

Through the MemoryPlugin interface:
1. Click the MemoryPlugin button
2. Select your active bucket from the dropdown
3. Memories from that bucket will be used
4. Switch buckets for different contexts

## Limitations and Workarounds

### Current Limitations
- No automatic memory loading (requires button click)
- No native Gemini API for memories
- Mobile app doesn't support extensions

### Workarounds
- Create memory templates for quick injection
- Use keyboard shortcuts where available
- Set up routine memory review sessions
- Consider MCP-compatible alternatives for automation
