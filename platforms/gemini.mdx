Use MemoryPlugin with Google Gemini at [gemini.google.com](https://gemini.google.com).

## How to Use

### Browser Extension
1. Install the [MemoryPlugin Browser Extension](/integrations/browser-extension)
2. Go to [gemini.google.com](https://gemini.google.com)
3. Click the MemoryPlugin button to inject memories
4. Start chatting with memory context

## Chat History

You can sync your Gemini conversation history into MemoryPlugin using the browser extension's online sync feature. Once synced, your past Gemini conversations become searchable and available as context across all your AI tools.

See the [Chat History](/features/chat-history/introduction) docs for more details.
