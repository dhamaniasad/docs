## Overview

Delete a single memory by its ID. The memory is removed from both the database and the vector store.

### Example

```bash
curl -X DELETE https://www.memoryplugin.com/api/memories/mem-abc-123 \
  -H "Authorization: Bearer YOUR_API_KEY"
```
