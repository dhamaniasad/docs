## OpenAPI Specification

Our API is fully documented using the industry-standard OpenAPI format:

## Authentication

All API requests require authentication using a Bearer token:

```http
Authorization: Bearer YOUR_API_TOKEN_HERE
```

### Getting Your API Token

<Steps>
  <Step title="Access Dashboard">
    Go to your [MemoryPlugin Dashboard](https://memoryplugin.com/dashboard)
  </Step>
  <Step title="Find API Section">
    Look for the API or Auth Token section
  </Step>
  <Step title="Copy Token">
    Copy your authentication token
  </Step>
  <Step title="Keep Secure">
    Treat this token like a password - keep it secure and regenerate if compromised
  </Step>
</Steps>

## Core API Endpoints

The MemoryPlugin API provides endpoints for:

### Memory Management
- **GET /memories** - Retrieve memories
- **POST /memories** - Create new memories
- **PUT /memories/{id}** - Update existing memories
- **DELETE /memories/{id}** - Delete memories

### Bucket Management
- **GET /buckets** - List memory buckets
- **POST /buckets** - Create new buckets
- **PUT /buckets/{id}** - Update bucket details
- **DELETE /buckets/{id}** - Delete buckets

### Search & Retrieval
- **GET /memories/search** - Search through memories
- **GET /memories/recent** - Get recently updated memories
- **GET /buckets/{id}/memories** - Get memories from specific bucket

## Example Integration Code

Here's a simple example of how to interact with the MemoryPlugin API:

<CodeGroup>

```javascript JavaScript/Node.js
const axios = require('axios');

const client = axios.create({
  baseURL: 'https://api.memoryplugin.com',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN_HERE',
    'Content-Type': 'application/json'
  }
});

// Get all memories
async function getMemories() {
  try {
    const response = await client.get('/memories');
    return response.data;
  } catch (error) {
    console.error('Error fetching memories:', error);
  }
}

// Create a new memory
async function createMemory(content, bucketId = null) {
  try {
    const response = await client.post('/memories', {
      content: content,
      bucketId: bucketId
    });
    return response.data;
  } catch (error) {
    console.error('Error creating memory:', error);
  }
}
```

```python Python
import requests

class MemoryPluginClient:
    def __init__(self, api_token):
        self.base_url = 'https://api.memoryplugin.com'
        self.headers = {
            'Authorization': f'Bearer {api_token}',
            'Content-Type': 'application/json'
        }
    
    def get_memories(self):
        response = requests.get(
            f'{self.base_url}/memories',
            headers=self.headers
        )
        return response.json()
    
    def create_memory(self, content, bucket_id=None):
        data = {'content': content}
        if bucket_id:
            data['bucketId'] = bucket_id
            
        response = requests.post(
            f'{self.base_url}/memories',
            json=data,
            headers=self.headers
        )
        return response.json()

# Usage
client = MemoryPluginClient('YOUR_TOKEN_HERE')
memories = client.get_memories()
```

```curl cURL
# Get all memories
curl -X GET "https://api.memoryplugin.com/memories" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json"

# Create a new memory
curl -X POST "https://api.memoryplugin.com/memories" \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Remember that the user prefers technical explanations",
    "bucketId": "bucket-id-here"
  }'
```
